Introduction

 

KML creator allows you to convert comma separated variable (CSV) GPS data into a KML file for viewing in Google Earth.

A Basic KML simply displays track information in Google Earth.

An Advanced KML provides the capability to display data as traces in the Elevation Profile function in Google Earth

Creating A Basic KML

 

1.       Select  File > Open File

2.      Input the number of the header row for your CSV file.

Header row is where the titles are in the CSV. In the example CSV (below) the Header Row is = 4.

3.       Using the drop down menus, select the columns which corresponds to Latitude, Longitude and Altitude.

 

Note: Latitude and Longitude must be in degrees decimal (for example 48.635645)

4.      Select Meters or Feet using the radio buttons.

5.      If  desired select the path and fill colours as well as  the opacity (if  you are boring  you may leave as the default colours)

At any stage if you are aware of any erroneous data rows you can delete them using the Delete Rows button

 

6.       Click Create Basic KML. (Large files may take up to 5 minutes depending on your computers performance)

 

Creating An Advanced KML

 

1.       Follow the first 5 steps from Creating a Basic KML procedure

  1. Click Create Advanced KML button
  2. Using the dropdowns select the data sets to be displayed.
  3. Click OK to create Advanced KML. (Large files may take up to 5 minutes depending on your computers performance)

Viewing Data in Google Earth

1.       Open the KML file that was created using the KML Creator.

2.       On the left hand side under the Places title, a place named Flight Profile will be displayed.

3.       Click on the + sign to explore the flight path.

4.        Right click on the Flight Profile folder and select Show Elevation Profile, a graph will be displayed underneath the Google Earth map.

5.       Two data sets can be displayed at any one time, to switch between data sets, click on their titles above the graph.

System Requirements

The following software applications are required to use the KML Creator and the files it generates:

·         Microsoft .Net Framework 3.5 or greater.

·         Google Earth 6.0 or greater.
