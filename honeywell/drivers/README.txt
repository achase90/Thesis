******************************
BEFORE PLUGGING USB DEVICE IN
******************************
INSTALLATION:
Copy the contents of this zipped file to the directory:
C:\Honeywell\USB_Driver_Install\
Run "PreInstaller.exe" 

******************************
AFTER PLUGGING USB DEVICE IN
******************************
VERIFY Installation:
Right click "My Computer"
Select "Properties"
Select "Hardware" tab
Click on "Device Manager"
Expand "Ports"
Honeywell USB device should be listed as one of the ports
